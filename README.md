# CHIP-8 Emulator

![Deno](https://img.shields.io/badge/Deno-2.x-black?logo=deno)
![TypeScript](https://img.shields.io/badge/TypeScript-5.x-blue?logo=typescript)
![License](https://img.shields.io/badge/License-MIT-green)
![Built with Claude](https://img.shields.io/badge/Built%20with-Claude%20Code-orange?logo=anthropic)

A fully-featured CHIP-8 emulator written in TypeScript, running on Deno, and playable in any modern browser. CHIP-8 is a virtual machine from the 1970s originally designed to simplify game development on early microcomputers; today it serves as the classic "hello world" of emulator development. This implementation was built entirely with [Claude Code](https://claude.ai/claude-code) by Anthropic.

## Features

- **35 CHIP-8 opcodes** fully implemented
- **64×32 monochrome display** rendered at 10× scale (640×320 canvas)
- **Adjustable CPU speed**: 50–1000 Hz via slider (default 500 Hz)
- **60 Hz delay & sound timers**
- **Web Audio API buzzer** — 440 Hz square wave
- **ROM loading** from local files (`.ch8` / `.rom` / `.bin`) or URL
- **23 CPU opcode unit tests** (all passing)
- **Dark-themed browser UI** with Start / Pause / Reset controls

## Getting Started

**Prerequisites:** [Deno 2.x](https://deno.land/#installation) installed.

```bash
git clone <repo-url>
cd chip8
deno task dev   # build bundle + start server
# Open http://localhost:8080/public/index.html
```

Or step by step:

```bash
deno task build   # bundles src/emulator.ts → public/bundle.js
deno task serve   # starts HTTP server on port 8080
```

## Implementation Notes

A few deliberate compatibility choices that affect ROM behavior:

- **`8XY6` / `8XYE` (shift):** operates on `Vx` directly, ignoring `Vy` — modern/CHIP-48 behavior for maximum ROM compatibility.
- **`FX55` / `FX65` (store/load):** does **not** increment `I` after operation — CHIP-48 behavior expected by most modern ROMs.
- **`FX0A` (wait for key):** uses a `halted` flag; the CPU freezes and resumes transparently on the next `step()` call after any key is pressed.
- **Game loop:** fixed-timestep accumulator driven by `requestAnimationFrame` decouples CPU speed from display refresh rate.

## Compatible ROMs

Any public-domain CHIP-8 ROM in `.ch8` format should work. Good starting points for testing:

- **IBM Logo** — classic display test
- **Corax89 opcode test** — comprehensive opcode coverage checker
- **BC_Test** — behavioral compatibility test
